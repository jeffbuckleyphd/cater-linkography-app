# Linkography Web App

The Linkography Web App is an open research tool for visualising and analysing linkographic data. It is intended to support researchers, educators, and students who are interested in examining the structure of dialogue, design activity, collaborative work, or other forms of sequenced interaction.

The app was developed by **Nicolaas Blom** and **Jeffrey Buckley** as a research resource associated with the **Centre for the Advancement of Technology Education Research (CATER)**.

## Use the app

The recommended way to use the app is through the browser-based version linked from the CATER website.

Users do not need to install R, RStudio, or any additional software to use the browser version. The app runs in the browser using Shinylive, which allows a Shiny app to be provided as a static web resource.

## Purpose

The app supports the visualisation and analysis of linkographic data. It can be used to explore links between moves, link density, participant contributions, and structural features of interaction.

The app is provided as a methodological resource. Users are responsible for checking that their data are formatted appropriately and that outputs are interpreted in relation to their own research context.

## Open use

This app is provided for open use by the research and education community. Users are welcome to use the app for research, teaching, supervision, workshops, and methodological training.

The source code is made available so that others can inspect, adapt, and extend the app. If the app is adapted or reused in another project, acknowledgement of the original developers is appreciated.

## Data and privacy

The browser-based version is intended to run locally in the user's browser. Users should still avoid uploading sensitive, confidential, or identifiable data unless they have confirmed that doing so is appropriate for their context.

Where possible, users should anonymise or pseudonymise data before using the app.

## Source code

The source code for this app is available in this repository. The app has been exported using Shinylive so that it can be accessed through a browser without requiring a hosted Shiny server.
